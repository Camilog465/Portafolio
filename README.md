## Conceptos trabajados en este proyecto
A continuacion los siguientes conceptos son los que se implemetaron en este proyecto 

- BEM: Metodologia CSS

- MOBILE FIRST: Metodologia de desarrollo

- Menu desplegable
- Formulario, guarda datos en LocalStorage
